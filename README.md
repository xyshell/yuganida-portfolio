# yuganida-portfolio
Yuganida's portfolio website, published at [https://yuganida.github.io](https://yuganida.github.io/)


## Quick Start

### download npm:

https://nodejs.org/en/download

```
npm version
```

### install hexo and init

```
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
```

### git deploy

```
npm install hexo-deployer-git --save
```
